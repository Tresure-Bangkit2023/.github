# System Requirement Specification
This is a repo for breaking down the desired system requirement.

## Data Design
Below are the defined model to implement on the application:

### User
Attributes:
- id
- username
- password
- email
- full_name

### Place
- id
- name
- description
- price
- coordinate
- image

### Place_Rating
- user_id
- place_id
- rating
